# Приложение «Movie Library»

**Автор:** Амин Мазитов

## Описание
Программа для ведения личного списка фильмов. Позволяет добавлять информацию о кинокартинах, сохранять её в JSON-файл и фильтровать список по жанру или году выпуска.

## Функционал
- Добавление фильма (Название, Жанр, Год, Рейтинг).
- Валидация: рейтинг строго 0-10, год — целое число.
- Автоматическое сохранение в `movies.json`.
- Фильтрация по жанру (частичное совпадение) и году.

## Инструкция по использованию
1. Запустите файл: `python movie_library.py`.
2. Введите данные в верхние поля и нажмите «Добавить».
3. Для поиска введите критерии в секцию «Фильтрация» и нажмите «Применить».
4. Чтобы вернуть полный список, нажмите «Сброс».

## Примеры тестов
- **Тест на ошибки:** Введите в поле рейтинга «12» или «текст». Программа выдаст предупреждение.
- **Тест фильтра:** Добавьте фильмы с жанрами «Комедия» и «Драма». Введите «ком» в фильтр — останется только комедия.
